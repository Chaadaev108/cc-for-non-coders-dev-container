# Документация проекта: cc-for-non-coders-dev-container

**Дата генерации:** 2026-02-28
**Workflow:** document-project v1.2.0, режим initial_scan (deep)

## Обзор проекта

Docker-контейнер с полностью настроенной учебной средой для курса **«Claude Code для непрограммистов»**. Предоставляет облачную IDE, файловый менеджер и AI-ассистента через единый веб-интерфейс с авторизацией.

### Краткая справка

| Параметр | Значение |
|----------|---------|
| **Тип** | Инфраструктурный (Docker) — монолит |
| **Стек** | Docker, Ubuntu 22.04, Python 3, Node.js 22, code-server |
| **Язык контента** | Русский |
| **API-провайдер** | Z.AI (api.z.ai) |
| **Модели** | GLM-5 (Opus/Sonnet), GLM-4.5-Air (Haiku) |
| **Auth** | HMAC-SHA256 cookie |
| **Порт** | 8080 |

### Архитектура (одно предложение)

Python HTTP-прокси с авторизацией (`auth-gateway.py` :8080) проксирует запросы к code-server (:8081) и File Browser (:9090) внутри одного контейнера.

## Быстрый старт

```bash
# 1. Клонировать и настроить
git clone <url> && cd cc-for-non-coders-dev-container
cp .env.example .env
# Отредактировать .env: PASSWORD и GLM_API_KEY

# 2. Собрать и запустить
./run.sh

# 3. Открыть в браузере
open http://localhost:8080
```
