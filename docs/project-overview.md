# Обзор проекта: cc-for-non-coders-dev-container

**Дата генерации:** 2026-02-28
**Тип проекта:** Инфраструктурный (Docker-контейнер) + веб-элементы
**Репозиторий:** Монолит

## Назначение

Docker-контейнер, предоставляющий полностью настроенную учебную среду для курса **«Claude Code для непрограммистов»**. Студент получает доступ к облачной IDE (code-server), файловому менеджеру (File Browser) и терминалу с предустановленным Claude Code CLI — через единую веб-точку входа с авторизацией.

### Для кого этот проект

**Непрограммисты** (предприниматели, менеджеры, аналитики, маркетологи) — люди, которые хотят использовать AI-ассистента для повседневных рабочих задач: обработки документов, создания отчётов, анализа данных, автоматизации рутины.

### Что даёт контейнер

- Готовую рабочую среду без установки чего-либо на своём компьютере
- 28 демо-проектов, разбитых по 5 учебным сессиям
- 19 предустановленных навыков (skills) для Claude Code
- Все необходимые инструменты: LibreOffice, FFmpeg, Pandoc, Playwright, OCR и другие

## Архитектурный паттерн

**Reverse-proxy gateway** — единый Python HTTP-сервер (`auth-gateway.py`) проксирует запросы к двум бэкенд-сервисам внутри контейнера:

```
Браузер → :8080 (auth-gateway.py)
              ├── /ide/*    → code-server :8081  (WebSocket + HTTP)
              └── /files/*  → File Browser :9090  (HTTP)
```
